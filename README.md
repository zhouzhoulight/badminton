# 一拍即慧项目展示网站

这是一个面向 GitHub Pages 的静态单页项目展示网站，内容来自项目申请书并根据当前需求整理为中文展示版本。

## 文件结构

- `index.html`：页面结构与项目内容
- `styles.css`：视觉样式与响应式布局
- `script.js`：技术路线切换、系统演示、周进度、团队筛选等交互
- `assets/icons/logo.svg`：项目 Logo
- `assets/images/hero-ai-badminton.png`：首页视觉图

## 部署到 GitHub Pages

把本目录内容推送到 GitHub 仓库后，在仓库的 `Settings -> Pages` 中选择对应分支和根目录即可发布。
